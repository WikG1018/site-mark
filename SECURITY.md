# Security Policy

## Supported versions

Security fixes are provided for the latest tagged SiteMark release.

## Reporting

Do not publish vulnerabilities, signing material, or private engineering photos
in a public issue. Contact the repository owner privately through the security
reporting channel configured on GitHub.

SiteMark's original-photo SHA-256 value is intended for local traceability. It
does not make a rooted device, a modified build, or an exported CSV tamper-proof
and must not be represented as forensic certification.
