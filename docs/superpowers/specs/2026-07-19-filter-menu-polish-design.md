# 筛选菜单与项目水印设置文案优化设计

## 目标

- 解决 360dp 等窄屏下筛选按钮文字与下拉箭头重叠的问题。
- 让筛选菜单的展开、选中状态和层次更清晰。
- 将项目级水印设置明确命名为“此项目水印设置”。

## 界面设计

筛选按钮移除右侧箭头，保留 44dp 高度、10dp 圆角及文字水平、垂直居中。按钮边框和可点击状态已经能够表达筛选入口，移除箭头可以在四等分窄屏布局中完整显示“全部项目”“全部年份”等文字。

展开菜单使用圆角卡片、轻阴影及 Flutter 内置的短促淡入展开动画。菜单项保持易点击高度；当前选项使用主题浅色背景与勾选图标标识，其他选项保持简洁。月份、日期未满足上级筛选条件时继续保持禁用状态。

项目详情页入口提示及项目水印设置页标题使用新的项目专用文案：中文“此项目水印设置”，英文“Project watermark settings”。通用“水印设置”文案不作无关修改。

## 验收标准

- 360dp 宽度下四个筛选按钮保持同一行，文字与任何图标均不重叠。
- 筛选按钮文字完整、居中，禁用状态清晰。
- 菜单展开和收起动画自然，菜单为圆角浮层且当前选项有勾选和背景提示。
- 选择项目、年份、月份或日期后，原有级联筛选逻辑保持不变。
- 项目详情入口及设置页标题显示“此项目水印设置”。
- 新增窄屏布局、菜单选中状态及项目专用文案的回归测试。
